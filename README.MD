# EBAS

A project made with love and care by [m3101](https://github.com/m3101)

---

EBAS is an abstract rule system that can be used to implement/describe a multitude of games. It has a companion project: SEBAS (this repository), which allows for effortlessly implementing EBAS-based sytems as computer games.

More information is available in the [wiki](https://github.com/m3101/1AP.009_EBAS/wiki).
